# 城市密集建筑群 InSAR 监测可用性量化评估与安全风险分级研究

## 项目简介
本项目用于完成毕业设计，研究内容围绕城市密集建筑群中 InSAR 监测可用性评估、建筑几何指标提取以及建筑安全风险分级展开。

## 当前目标
1. 构建建筑几何属性—InSAR 监测质量—时序形变综合数据库
2. 提取建筑级几何指标与街区级形态指标
3. 分析城市形态对 InSAR 监测可用性的影响
4. 建立建筑安全风险分级模型
5. 输出专题图、论文与答辩材料
